Create a keyboard navigation hook adapted from `useColumnNavigation.ts` but simplified for list view (no left/right column navigation).

**Props interface:**
```typescript
interface UseListNavigationProps {
  itemCount: number;
  focusedIndex: number;
  itemNames: string[];
  virtualizer: Virtualizer<HTMLDivElement, Element> | null;
  onFocusChange: (index: number) => void;
  onSelect: (index: number) => void;
  onOpen: (index: number) => void;      // Enter key -- open folder or select file
  onNavigateUp: () => void;             // Backspace -- go to parent directory
}
```

**Return type:** `{ handleKeyDown: (e: React.KeyboardEvent) => void }`

**Key handlers:**

- **ArrowDown**: `e.preventDefault()`. If `focusedIndex < itemCount - 1`, increment focus, call `onSelect(newIndex)`, scroll virtualizer to new index with `align: 'auto'`.
- **ArrowUp**: `e.preventDefault()`. If `focusedIndex > 0`, decrement focus, call `onSelect(newIndex)`, scroll virtualizer to new index.
- **Enter**: `e.preventDefault()`. If `focusedIndex >= 0`, call `onOpen(focusedIndex)`.
- **Backspace**: `e.preventDefault()`. Call `onNavigateUp()`.
- **Default (printable character)**: Type-ahead search. Same logic as `useColumnNavigation.ts` lines 75-118. Accumulate typed characters in a ref, find first matching entry name (case-insensitive startsWith), clear after 500ms timeout. On match, call `onFocusChange(matchIndex)`, `onSelect(matchIndex)`, scroll virtualizer to match.

Import `Virtualizer` type from `@tanstack/react-virtual`. Use `useCallback` for `handleKeyDown` and `useRef` for search state, same as useColumnNavigation.

Named export: `export function useListNavigation(...)`.

**1. Create `src/renderer/components/ListView/ListView.tsx`:**

This is the main container. It must be a drop-in replacement for ColumnView in terms of callback interface (Phase 17 compatibility).

**Props interface:**
```typescript
interface ListViewProps {
  serverId: string;
  initialPath?: string;
  navigateTo?: string | null;
  showHidden?: boolean;
  onFileSelect?: (file: FileEntry) => void;
  onPathChange?: (path: string) => void;
  onNavigationComplete?: () => void;
  onRefreshColumn?: (refreshFn: () => void) => void;
  onFavoritesChanged?: () => void;
  onMoveToClick?: (file: FileEntry) => void;
  onFilesLoaded?: (files: FileEntry[]) => void;
}
```

Note: `onFileSelect` takes `(file: FileEntry)` without `columnIndex` -- list view has no column concept. Phase 17 will align the interface.

**State:**
- `currentPath: string` -- initialized from `initialPath ?? '/'`
- `entries: FileEntry[]` -- raw fetched entries
- `loading: boolean`
- `error: string | null`
- `sortState: SortState` -- initialized to `{ column: 'name', direction: 'asc' }`. **Critical: this state is NOT reset on directory change** (SORT-04).
- `selectedIndex: number` -- currently selected row index (-1 = none)
- `focusedIndex: number` -- keyboard focus position (-1 = none)

**Refs:**
- `scrollRef: React.RefObject<HTMLDivElement>` -- scroll container for virtualizer

**Directory fetching:**
- `fetchDirectory(path: string)` callback (useCallback, depends on serverId):
  1. Set loading=true, error=null
  2. Call `window.electronAPI.listDirectory(serverId, path)`
  3. Map entries: convert `modified` from string to `new Date(entry.modified)` (Pitfall: IPC dates)
  4. Filter hidden files if `!showHidden`: `entries.filter(e => !e.name.startsWith('.'))`
  5. Set entries, loading=false
  6. Call `onFilesLoaded?.(entries)` after setting state (Pitfall 6: lightbox needs this)
  7. On error: set error message, entries=[]
- useEffect: fetch when `currentPath` or `showHidden` changes (same pattern as ColumnView.tsx lines 350-365 and DirectoryList.tsx lines 59-72)
- useEffect: handle `navigateTo` prop for external navigation (PathBar clicks). When `navigateTo` changes and is non-null, set `currentPath = navigateTo`, then call `onNavigationComplete?.()`.

**Sorting:**
- `sortEntries` function (can be defined outside component or as useMemo):
  - SORT-03: Always sort directories before files (`a.isDirectory !== b.isDirectory` -> directory first)
  - Then sort by `sortState.column`: name (localeCompare), size (numeric), modified (getTime), kind (getFileKind(name, isDir).localeCompare)
  - Apply `sortState.direction` (asc/desc)
- `sortedEntries = useMemo(() => sortEntries(entries, sortState), [entries, sortState])`
- `handleSort(column: SortColumn)` callback: If same column, flip direction. Else set new column + 'asc'. Same pattern as DirectoryList.tsx lines 110-119.

**Virtualization:**
```typescript
const virtualizer = useVirtualizer({
  count: sortedEntries.length,
  getScrollElement: () => scrollRef.current,
  estimateSize: () => 32,
  overscan: 15,
});
```

**Keyboard navigation:**
```typescript
const itemNames = useMemo(() => sortedEntries.map(e => e.name), [sortedEntries]);

const { handleKeyDown } = useListNavigation({
  itemCount: sortedEntries.length,
  focusedIndex,
  itemNames,
  virtualizer,
  onFocusChange: setFocusedIndex,
  onSelect: (index) => {
    setSelectedIndex(index);
    setFocusedIndex(index);
    const file = sortedEntries[index];
    if (file) onFileSelect?.(file);
  },
  onOpen: (index) => {
    const file = sortedEntries[index];
    if (!file) return;
    if (file.isDirectory) {
      handleDirectoryOpen(file.path);
    } else {
      onFileSelect?.(file);
    }
  },
  onNavigateUp: handleNavigateUp,
});
```

**Directory navigation:**
- `handleDirectoryOpen(path: string)`: Set currentPath, reset selectedIndex to -1 and focusedIndex to 0, call `onPathChange?.(path)`.
- `handleNavigateUp()`: If currentPath === '/', return. Compute parent: `currentPath.substring(0, currentPath.lastIndexOf('/')) || '/'`. Call handleDirectoryOpen(parent).
- Double-click handler: For directories, call handleDirectoryOpen(file.path). For files, call onFileSelect?.(file).

**Row click handler:**
- `handleRowClick(index, e)`: Set selectedIndex and focusedIndex, call `onFileSelect?.(sortedEntries[index])`.

**Refresh function:**
- Expose a `refresh` function via `onRefreshColumn` callback: `onRefreshColumn?.(() => fetchDirectory(currentPath))`. Use useEffect to register this on mount and when currentPath changes.

**Render structure:**
```jsx
<div className="list-view" onKeyDown={handleKeyDown} tabIndex={0}>
  <ListHeader sort={sortState} onSort={handleSort} />
  <div className="list-view__scroll" ref={scrollRef}>
    {loading && <div className="list-view__loading">Loading...</div>}
    {error && <div className="list-view__empty">{error}</div>}
    {!loading && !error && sortedEntries.length === 0 && (
      <div className="list-view__empty">Empty directory</div>
    )}
    {!loading && !error && sortedEntries.length > 0 && (
      <div style={{ height: virtualizer.getTotalSize(), position: 'relative', width: '100%' }}>
        {virtualizer.getVirtualItems().map((virtualRow) => {
          const entry = sortedEntries[virtualRow.index];
          return (
            <div
              key={entry.path}
              style={{
                position: 'absolute',
                top: 0,
                left: 0,
                width: '100%',
                transform: `translateY(${virtualRow.start}px)`,
              }}
            >
              <ListRow
                file={entry}
                isSelected={virtualRow.index === selectedIndex}
                isFocused={virtualRow.index === focusedIndex}
                serverId={serverId}
                showHiddenFiles={showHidden ?? false}
                onRefresh={() => fetchDirectory(currentPath)}
                onFavoritesChanged={onFavoritesChanged}
                onMoveToClick={onMoveToClick}
                onClick={(e) => handleRowClick(virtualRow.index, e)}
                onDoubleClick={() => {
                  if (entry.isDirectory) {
                    handleDirectoryOpen(entry.path);
                  } else {
                    onFileSelect?.(entry);
                  }
                }}
              />
            </div>
          );
        })}
      </div>
    )}
  </div>
</div>
```
