<task type="auto">
  <name>Task 1: Add PDF type to shared types</name>
  <files>src/shared/types.ts, src/preload/preload.ts</files>
  <action>
1. In `src/shared/types.ts`, add a new variant to the `PreviewData` union type:
   ```typescript
   | { type: 'pdf'; dataUrl: string; pageCount: number; fileSize: number; isLarge: boolean }
   ```
   - `dataUrl`: base64-encoded PDF data for react-pdf
   - `pageCount`: total pages (extracted from PDF.js or set to 0 if unknown initially)
   - `fileSize`: file size in bytes
   - `isLarge`: true if pageCount > 100 (per CONTEXT.md: warn on 100+ pages)

2. In `src/preload/preload.ts`, update the duplicated `PreviewData` type to match (preload has its own copy for isolation):
   ```typescript
   | { type: 'pdf'; dataUrl: string; pageCount: number; fileSize: number; isLarge: boolean }
   ```

Note: pageCount will be 0 from backend (we can't easily parse PDF page count in Node without PDF.js). The renderer will determine actual page count when loading with react-pdf.
  </action>
  <verify>Run `npm run typecheck` - no type errors related to PreviewData</verify>
  <done>PreviewData union includes pdf variant in both shared types and preload</done>
</task>

  <action>
1. In the `detectFileType` function, add PDF detection BEFORE the binary default case:
   ```typescript
   // PDF extension
   if (ext === 'pdf') {
     return { category: 'pdf', mimeType: 'application/pdf' };
   }
   ```
   Note: Add 'pdf' to the FileTypeInfo category union if needed.

2. Update the `FileTypeInfo` interface in `src/shared/types.ts` to include 'pdf' category:
   ```typescript
   category: 'image' | 'code' | 'text' | 'binary' | 'pdf';
   ```

3. In the `preview:read-file` handler, add a case for PDF category after image handling and before code:
   ```typescript
   // Handle PDF files
   if (fileType.category === 'pdf') {
     const dataUrl = `data:application/pdf;base64,${buffer.toString('base64')}`;
     return {
       type: 'pdf',
       dataUrl,
       pageCount: 0, // Will be determined by renderer with react-pdf
       fileSize: stats.size,
       isLarge: false, // Will be determined by renderer after loading
     } as PreviewData;
   }
   ```

4. Also handle PDF for cached files (in the cache hit section):
   ```typescript
   if (fileType.category === 'pdf') {

     return {
       type: 'pdf',
       dataUrl,
       pageCount: 0,
       fileSize: cached.size,
       isLarge: false,
     };
   }
   ```
