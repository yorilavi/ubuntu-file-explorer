  <action>
Create `src/renderer/utils/` directory (if it doesn't exist) and two utility files.

**formatters.ts** -- Three pure functions:

1. `formatSize(bytes: number): string`
   - Use threshold-based if/else chain (NOT Math.log approach -- the threshold version is clearer and avoids floating-point edge cases)
   - 0 -> "0 B", <1024 -> "{n} B", <1MB -> "{n.toFixed(1)} KB", <1GB -> "{n.toFixed(1)} MB", else -> "{n.toFixed(1)} GB"
   - This replaces: FileItem.tsx `formatBytes`, FileRow.tsx `formatSize`, PreviewPanel.tsx `formatSize`, ImagePreview.tsx `formatSize`, PDFPreview.tsx `formatSize`

2. `formatDate(date: Date | string): string`
   - Accept both Date objects AND string (IPC serializes Date to string)
   - Convert string to Date if needed: `const d = date instanceof Date ? date : new Date(date)`
   - Return empty string for invalid/undefined dates (guard: `if (!date) return ''`)
   - Use `Intl.DateTimeFormat` or `toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric', hour: 'numeric', minute: '2-digit' })`
   - Output example: "Jan 15, 2026, 3:42 PM"
   - This replaces: FileRow.tsx `formatDate`, ImagePreview.tsx `formatDate`

3. `formatPermissions(mode: string): string`
   - Convert octal string to rwx: parse as octal int, extract 3 groups of 3 bits
   - '0755' -> 'rwxr-xr-x', '0644' -> 'rw-r--r--'
   - This replaces: FileRow.tsx `formatPermissions`

**fileKinds.ts** -- Extension-to-kind label mapping:

1. `FILE_KIND_MAP: Record<string, string>` -- Map of lowercase extensions to human-readable labels covering: images (jpg, jpeg, png, gif, webp, svg, bmp, ico, avif), documents (pdf, doc, docx, xls, xlsx, ppt, pptx), code (js, jsx, ts, tsx, py, rb, go, rs, c, cpp, h, java, kt, swift, cs, php), web (html, htm, css, scss), data/config (json, yaml, yml, xml, toml, csv), text (md, mdx, txt, log), shell (sh, bash, zsh), archives (zip, tar, gz, 7z, rar), media (mp3, wav, flac, mp4, mkv, avi), config (ini, conf, cfg, env), sql, dockerfile

2. `KNOWN_FILENAMES: Record<string, string>` -- Map for extensionless files: Makefile -> "Makefile", Dockerfile -> "Dockerfile", LICENSE -> "License", Vagrantfile -> "Vagrantfile", Procfile -> "Procfile". Also handle dotfiles: .gitignore -> "Git Ignore", .gitattributes -> "Git Attributes", .editorconfig -> "EditorConfig", .prettierrc -> "Prettier Config", .eslintrc -> "ESLint Config", .dockerignore -> "Docker Ignore", .env -> "Environment File"

3. `getFileKind(fileName: string, isDirectory: boolean): string`
   - If `isDirectory` -> return "Folder"
   - Check `KNOWN_FILENAMES` for exact filename match (case-insensitive for extensionless, exact for dotfiles)
   - Extract extension: `fileName.lastIndexOf('.')` to split (use lastIndexOf, NOT split('.').pop() which fails for files like 'archive.tar.gz')
   - Lookup extension in `FILE_KIND_MAP`
   - Fallback: "Document"

All functions are exported individually (named exports, no default export).
