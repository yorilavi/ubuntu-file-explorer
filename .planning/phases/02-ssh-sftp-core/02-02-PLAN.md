Create src/main/ssh/ssh-service.ts that manages SSH connections.

The service maintains a Map of active connections keyed by server ID.

```typescript
import { Client, type ConnectConfig } from 'ssh2';
import { readFileSync } from 'fs';
import type { Server, ConnectionState } from './types';
```

1. **Connection storage:**
   ```typescript
   const connections = new Map<string, Client>();
   const connectionStates = new Map<string, ConnectionState>();
   ```

2. **State change callback:**
   ```typescript
   type StateCallback = (serverId: string, state: ConnectionState) => void;
   let stateCallback: StateCallback | null = null;
   export function setStateCallback(cb: StateCallback): void { stateCallback = cb; }
   ```

3. **connectSSH(server: Server, password?: string): Promise<void>**
   - Build ConnectConfig from Server:
     - host: server.host
     - port: server.port
     - username: server.username
     - For authMethod 'key': privateKey from readFileSync(server.keyPath!), passphrase if provided
     - For authMethod 'agent': agent: process.env.SSH_AUTH_SOCK
     - For authMethod 'password': password from parameter
   - Create new Client()
   - Wrap in Promise with event handlers:
     - 'ready' -> resolve, update state to { status: 'ready' }
     - 'error' -> reject, update state to { status: 'error', message: err.message }
     - 'close' -> cleanup, update state to { status: 'idle' }
   - Update state to { status: 'resolving' } before connect
   - Update state to { status: 'authenticating' } on connect call
   - Store client in connections Map on success

4. **disconnectSSH(serverId: string): void**
   - Get client from Map
   - Call client.end()
   - Remove from Map
   - Update state to { status: 'idle' }

5. **getConnection(serverId: string): Client | undefined**
   - Return from connections Map

6. **getConnectionState(serverId: string): ConnectionState**
   - Return from connectionStates Map, default { status: 'idle' }

Handle errors per RESEARCH.md pitfall #1: Always listen for 'error', 'close', 'end' events.

1. **registerSSHHandlers(mainWindow: BrowserWindow): void**
   Register all IPC handlers:

   - **'ssh:list-servers'**: Return combined list - getSSHConfigServers() + getAllConnections()

   - **'ssh:add-connection'**: Accept CustomConnection, generate ID if missing, saveConnection, optionally saveCredential if password provided

   - **'ssh:remove-connection'**: Accept id, deleteConnection, deleteCredential

   - **'ssh:connect'**: Accept serverId and optional password
     - Find server in combined list
     - If authMethod is 'password' and no password param, try getCredential
     - Call connectSSH
     - Return { success: true } or { success: false, error: message }

   - **'ssh:disconnect'**: Accept serverId, call disconnectSSH

   - **'ssh:list-directory'**: Accept serverId and path, call listDirectory

   - **'ssh:get-connection-state'**: Accept serverId, return getConnectionState

2. **Set up state callback** to send updates to renderer:
   ```typescript
   setStateCallback((serverId, state) => {
     mainWindow.webContents.send('ssh:state-change', serverId, state);
   });
   ```

Update src/preload/preload.ts to expose SSH API:
```typescript
const electronAPI = {
  // ... existing methods ...

  // SSH Operations
  listServers: (): Promise<Server[]> =>
    ipcRenderer.invoke('ssh:list-servers'),

  addConnection: (connection: Omit<CustomConnection, 'id' | 'source'>, password?: string): Promise<{ success: boolean; id?: string; error?: string }> =>
    ipcRenderer.invoke('ssh:add-connection', connection, password),

  removeConnection: (id: string): Promise<void> =>
    ipcRenderer.invoke('ssh:remove-connection', id),

  connect: (serverId: string, password?: string): Promise<{ success: boolean; error?: string }> =>
    ipcRenderer.invoke('ssh:connect', serverId, password),

  disconnect: (serverId: string): Promise<void> =>
    ipcRenderer.invoke('ssh:disconnect', serverId),

  listDirectory: (serverId: string, path: string): Promise<DirectoryListing> =>
    ipcRenderer.invoke('ssh:list-directory', serverId, path),

  getConnectionState: (serverId: string): Promise<ConnectionState> =>
    ipcRenderer.invoke('ssh:get-connection-state', serverId),

  onConnectionStateChange: (callback: (serverId: string, state: ConnectionState) => void): (() => void) => {
    const handler = (_event: unknown, serverId: string, state: ConnectionState) => callback(serverId, state);
    ipcRenderer.on('ssh:state-change', handler);
    return () => ipcRenderer.removeListener('ssh:state-change', handler);
  },
};
```
