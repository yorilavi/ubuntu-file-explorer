**RemoteFolderPicker.tsx** - Main modal component:
1. Props interface:
   - isOpen: boolean

   - serverId: string
   - sourcePath: string (full path of file being moved)
   - sourceFileName: string
   - onMoveConfirm: (destDir: string) => void

2. State:
   - selectedPath: string (defaults to source file's parent directory)
   - showHiddenFiles: boolean (load from getShowHiddenFiles IPC on mount)

3. Extract sourceDir from sourcePath (everything before last /)

4. Breadcrumb navigation:
   - Parse selectedPath into path segments
   - Render clickable breadcrumb for each segment
   - Clicking navigates (sets selectedPath and expands to that path)
   - Show "/" as root, then each folder name

5. Modal structure (follow AddConnectionModal pattern):
   - modal-overlay with backdrop click handler
   - modal container with max-width 500px, max-height 70vh

     - FolderTree (scrollable, ~350px height)

     - Left: "Move {fileName} to: {selectedPath}" (truncate if long)
