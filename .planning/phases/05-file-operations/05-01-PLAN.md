Create `src/main/ssh/file-operations-service.ts` with download and upload functions:

1. Import dependencies:
   - `createReadStream`, `createWriteStream` from 'fs'
   - `statSync` from 'fs' (for upload file size)
   - `path` (for basename)
   - `getSFTPWrapper` from './sftp-service'
   - `Stats` type from 'ssh2'

2. Implement `downloadFile`:
   ```typescript
   export async function downloadFile(
     serverId: string,
     remotePath: string,
     localPath: string,
     onProgress: (percent: number) => void
   ): Promise<void>
   ```
   - Get SFTP wrapper, throw "Not connected to server" if null
   - Get remote file stats via sftp.stat() for file size
   - Create read stream from sftp.createReadStream(remotePath)
   - Create write stream to local path via createWriteStream(localPath)
   - Track bytesTransferred on 'data' event, calculate percent, call onProgress
   - Pipe read to write stream
   - Return Promise that resolves on writeStream 'finish', rejects on error

3. Implement `uploadFile`:
   ```typescript
   export async function uploadFile(
     serverId: string,
     localPath: string,
     remoteDir: string,
     onProgress: (percent: number) => void
   ): Promise<string>  // Returns full remote path
   ```
   - Get SFTP wrapper, throw if null
   - Get local file size via statSync(localPath).size
   - Extract filename via path.basename(localPath)
   - Construct remotePath via path.posix.join(remoteDir, filename) - POSIX for remote!
   - Create read stream from local, write stream from sftp.createWriteStream(remotePath)
   - Track progress on 'data' event
   - Return remotePath on completion

Key patterns from RESEARCH.md:
- Use path.posix.join() for remote paths (POSIX format)
- Use path.join() for local paths (OS-aware)
- Stream-based with progress tracking

3. Implement `renameRemoteFile`:
   ```typescript
   export async function renameRemoteFile(
     serverId: string,
     oldPath: string,
     newName: string  // Just the new filename, not full path
   ): Promise<string>  // Returns new full path
   ```
   - Get SFTP wrapper, throw if null
   - Calculate newPath by replacing filename in oldPath:
     - `const dir = path.posix.dirname(oldPath)`
     - `const newPath = path.posix.join(dir, newName)`
   - Call sftp.rename(oldPath, newPath)
   - Return newPath

4. Implement `moveRemoteFile`:
   ```typescript
   export async function moveRemoteFile(
     serverId: string,
     sourcePath: string,
     destDir: string
   ): Promise<string>  // Returns new full path
   ```
   - Get SFTP wrapper, throw if null
   - Extract filename: `path.posix.basename(sourcePath)`
   - Construct destPath: `path.posix.join(destDir, filename)`
   - Call sftp.rename(sourcePath, destPath) - SFTP rename works for move
   - Return destPath

Important: All remote path operations use path.posix methods!
