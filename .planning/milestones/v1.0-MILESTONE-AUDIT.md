### Partial Requirement: FILE-05 (Move)

**Status:** Backend complete, UI deferred

**Rationale:** Native Electron dialogs (showOpenDialog) only browse local filesystem, not remote SFTP directories. Moving files requires a custom remote folder picker modal.

**Backend ready:**
- `moveRemoteFile` service function exists
- `file-ops:move` and `file-ops:move-with-picker` IPC handlers exist
- Preload exposes `moveFile` and `moveFileWithPicker` methods

**UI needed:** Custom RemoteFolderPicker modal component

## Cross-Phase Integration

### IPC Bridge Chain

```
preload.ts (API surface)
    ↓ ipcRenderer.invoke
main process handlers
    ↓ service calls
services (ssh, sftp, file-ops, favorites, etc.)
    ↓ external libs
ssh2, electron-conf, exifr, etc.
```

**Status:** ✓ All channels verified wired

### Data Flow

```
FileEntry → Column → FileItem → Context Menu
    ↓                              ↓
Preview Panel              File Operations
    ↓                              ↓
IPC handlers               Toast notifications
```

**Status:** ✓ All component chains verified

### State Management

| State | Location | Persistence |
|-------|----------|-------------|
| Connection state | App.tsx (connectionStates) | Session only |
| Selected server | App.tsx (selectedServer) | Session only |
| Column widths | ColumnView + IPC | electron-conf |
| Preview width | App.tsx + IPC | electron-conf |
| Favorites | useFavorites + IPC | electron-conf |
| Credentials | credential-store | safeStorage (encrypted) |

**Status:** ✓ All persistence verified

## E2E User Flows Verified

### Flow 1: Connect and Browse
1. App launches → Server list appears
2. Click server → Connection status shows
3. Connected → Root directory loads
4. Click folder → Opens in new column
5. Arrow keys → Navigate files

**Status:** ✓ Verified through Phase 2, 3 testing

### Flow 2: Preview Files
1. Select image → Preview appears in panel
2. Arrow keys → Preview updates (debounced)
3. Spacebar → Lightbox opens
4. Arrow keys in lightbox → Browse images
5. Escape → Close lightbox

**Status:** ✓ Verified through Phase 4, Post-v1 testing

### Flow 3: File Operations
1. Right-click file → Context menu appears
2. Download → Save dialog → Progress toast → Complete toast
3. Upload → Open dialog → Progress toast → Directory refreshes
4. Delete → Confirmation → Success toast
5. Rename → Inline edit → Enter saves

**Status:** ✓ Verified through Phase 5 testing

### Flow 4: Favorites
1. Right-click folder → "Add to Favorites"
2. Success toast → Appears in sidebar
3. Click favorite → Navigates to folder
4. Drag favorites → Reorder persists
5. App restart → Favorites preserved

**Status:** ✓ Verified through Phase 6 testing
