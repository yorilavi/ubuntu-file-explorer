# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-01-28
**Phases:** 1-6
**Total Plans:** 22

## Overview

This roadmap delivered a macOS Finder-like SSH file explorer in 6 phases. We started with Electron scaffolding and security foundations, then built SSH connectivity, followed by the core differentiators (Miller column navigation and instant previews), then file operations, and finally polish with favorites and error handling. Each phase delivered a verifiable capability that built toward the complete browsing experience.

## Milestone Summary

**Key Decisions:**
- Electron over Tauri/native (familiar web tech, faster to build)
- Column view over list/icon view (matches Finder mental model)
- Per-server favorites (servers have different use cases)
- electron-conf over electron-store (CommonJS/ESM compatibility)
- Custom column resize over react-resizable-panels (fixed min widths requirement)

**Issues Resolved:**
- ESM compatibility with @vitejs/plugin-react (downgraded to 4.x)
- TypeScript version incompatibility (upgraded to 5.5.0)
- CSP blocking data URLs for images
- Column resize library limitations
- Duplicate directory fetches on state updates

**Technical Debt:**
- Move file feature needs custom RemoteFolderPicker modal
- Folder upload needs tar/gzip + server-side extraction check
